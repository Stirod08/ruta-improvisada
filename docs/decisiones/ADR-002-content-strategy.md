# ADR-002

## Título

Estrategia de contenido basada en Markdown.

## Estado

Aceptado

## Contexto

El proyecto se enfoca en contenido turístico y SEO.

## Decisión

Todo el contenido será almacenado en archivos Markdown/MDX dentro del repositorio.

## Consecuencias Positivas

- Sin costos de CMS.
- Versionado completo.
- Alto rendimiento.
- Fácil despliegue.

## Consecuencias Negativas

- Requiere conocimientos básicos de Git para publicar contenido.
