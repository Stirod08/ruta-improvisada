# ADR-001

## Título

Selección de Astro como framework principal.

## Estado

Aceptado

## Contexto

El proyecto requiere:

- Alto rendimiento
- SEO
- Generación estática
- Contenido basado en Markdown

## Decisión

Se utilizará Astro como framework principal.

## Consecuencias

### Positivas

- Excelente SEO
- Muy rápido
- Fácil mantenimiento

### Negativas

- Menor ecosistema que Next.js
