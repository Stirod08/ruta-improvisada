# Principios Arquitectónicos

## Rendimiento Primero

Toda decisión debe priorizar velocidad de carga.

## SEO Primero

Toda página debe poder posicionarse en Google.

## Contenido Escalable

El contenido debe crecer sin modificar código.

## Mobile First

El diseño se construye primero para móviles.

## Accesibilidad

Toda funcionalidad debe ser usable desde teclado.
