# Convenciones

## Carpetas

Todas las carpetas usarán:

kebab-case

Ejemplo:

municipios-antioquia

## Archivos MDX

Todos los archivos de contenido usarán:

slug-del-contenido.mdx

Ejemplo:

jardin.mdx

guatape.mdx

salto-del-buey.mdx

## Componentes

PascalCase

HeroSection.astro
